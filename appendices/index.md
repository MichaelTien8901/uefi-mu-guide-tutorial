# Appendices

The appendices provide reference material that supports the main guide. Use them as a quick-lookup resource while working through the chapters or building your own firmware projects.

| Appendix | Topic |
|----------|-------|
| A | EDK2 Build System -- INF, DEC, DSC, FDF file formats and build commands |
| B | EDK2 to Project Mu Migration -- converting platforms, CI, and common pitfalls |
| C | Debugging Techniques -- DEBUG macro, GDB, QEMU monitor, serial output |
| D | References -- specifications, repositories, community resources |
| E | Glossary -- acronyms, error codes, terminology |
| F | Alternative Firmware -- coreboot, LinuxBoot, openSIL, oreboot |
